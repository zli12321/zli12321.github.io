abstract: >-
  We present MM-Zero, the first RL-based framework to achieve zero-data self-evolution for VLM reasoning. It introduces a multi-role self-evolving training framework comprising three specialized roles: a Proposer that generates abstract visual concepts and formulates questions; a Coder that translates these concepts into executable code (e.g., Python, SVG) to render visual images; and a Solver that performs multimodal reasoning over the generated images.
